Welcome to Dump!

Dump is a disposal solutions application which connects Houston area residents with depositories, recycling/donation centers, and landfills in their area.

Trying to 'dump' something?
Simply type the object in question and Dump will return a list of centers able to receive the item, in order of proximity to your current location

This project is on-going
